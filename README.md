# Test Version control

* This repository is created for the purpose of testing git remote version control functionalities
* It contains a text file named test.txt
* I used git and github to track changes and manage code history 

## Installation

### To clone this repository to your local machine, use the following command:

`https://github.com/doapo/Version-Control-with-Git-and-GitHub.git`

## Usage

### Making Changes:

Open test.txt and make your desired changes.

### Committing Changes:

`git add test.txt`

### Commit the changes:

`git commit -m "Your commit message"`

### Pushing Changes:

`git push origin main`

### Pulling Changes:

`git pull origin main`

### Create New Branch:
`git checkout -b "branch name"`

### Merge Branch:
`git merge "branch name"`
